# GitHub Upload Checklist

Upload the contents of `github_reproducibility_package` as the repository root.

## Upload

- `README.md`
- `UPLOAD_CHECKLIST.md`
- `requirements.txt`
- `experiment_config.json`
- `code/`
- `outputs/`
- `figures/`
- `docs/`

## Do Not Upload

- Reviewer comment files.
- Draft DOCX files.
- LaTeX auxiliary files such as `.aux`, `.log`, `.out`, `.bbl`, `.blg`, and `.synctex.gz`.
- Private notes, local cache folders, or non-anonymized files.
- The old `github_upload_package` folder, which contains earlier intermediate files.

## Manuscript Update

After repository creation, replace:

```text
https://github.com/USERNAME/c2-llm-agent-governance-simulator
```

with the actual GitHub address.
